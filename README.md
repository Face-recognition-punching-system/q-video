## 🚀 快速开始

### 环境

* Windows 10
* Visual Studio Code
* node.js 18.13.0
* npm 8.5.3

### 拉取代码

```
git clone https://github.com/Face-recognition-punching-system/q-video.git
```

### 安装依赖

```bash
npm install
```

### 调试

```bash
npm run start
```

### 打包

```bash
npm run build
```

### 测试

```bash
npm run test
```
